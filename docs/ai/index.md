# MinusAI

Servidor MCP (Model Context Protocol) que expõe ferramentas de IA para engenharia de software Delphi.

## Funcionalidades

- `explicar_codigo` — analisa arquivos `.pas` e extrai metadados
- `gerar_entidade` — gera entidade ORM (via template ou schema do banco)
- `criar_migracao` — gera arquivo de migração versionada
- `executar_consulta` — executa SQL contra banco FireDAC real
- `listar_tabelas` — lista tabelas do banco de dados
