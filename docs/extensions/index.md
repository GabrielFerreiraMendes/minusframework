# MinusExtensions

Pacote de extensões plugáveis para o MinusORM.

| Extensão | Descrição |
|----------|-----------|
| Soft Delete | Deleção lógica automática com `[DesabilitarSoftDelete]` |
| Auditoria | Quem criou, alterou e quando |
| Cache | Cache L2 distribuído |
| Bulk | Inserção/atualização em lote |
| Multi-Tenancy | Isolamento por tenant |
| JSON Serializer | Serializa entidades para JSON |
| Async | Operações assíncronas |
