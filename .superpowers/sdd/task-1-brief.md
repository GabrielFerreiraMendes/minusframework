### Task 1: Scaffold Docusaurus

**Files:**
- Create: `package.json`
- Create: `docusaurus.config.ts`
- Create: `sidebars.ts`
- Create: `tsconfig.json`
- Create: `.gitignore` (append Docusaurus entries)
- Create: `src/css/custom.css`

**Step 1:** Rename existing MkDocs docs/ to preserve content
```powershell
Move-Item docs docs-mkdocs
```

**Step 2:** Initialize Docusaurus in a temp dir then copy to root
```powershell
npx create-docusaurus@latest tmp-site --template classic --typescript --skip-install
Copy-Item tmp-site\* . -Recurse -Force
Remove-Item -Recurse -Force tmp-site
Remove-Item -Recurse -Force docs
npm install
```

**Step 3:** Configure docusaurus.config.ts (full code in plan)

**Step 4:** Create sidebars.ts
```typescript
import type {SidebarsConfig} from '@docusaurus/plugin-content-docs';

const sidebars: SidebarsConfig = {
  docsSidebar: [{type: 'autogenerated', dirName: '.'}],
};
export default sidebars;
```

**Step 5:** Create tsconfig.json
```json
{
  "extends": "@docusaurus/tsconfig",
  "compilerOptions": { "baseUrl": "." }
}
```

**Step 6:** Update .gitignore - append:
```
# Docusaurus
node_modules/
build/
.docusaurus/
.cache-loader/
```

**Step 7:** Run `npm install`

**Step 8:** Verify with `npx docusaurus build`

**Step 9:** Commit
